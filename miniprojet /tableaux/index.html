La vie multilingue des mots

Le projet vise à étudier l'usage d'un mot à travers différentes langues et de les comparer.

Les étapes du projet

Le projet se construit selon différentes étapes :

  1. choisir le mot ;
  2. choisir les URL pour l'étude multilingue de ce mot ;
  3. construire un corpus en récupérant le contenu textuel de ces pages ainsi que d'autres informations ;
  4. construire des nuages de mots ;
  5. construire des contextes autour de ces mots ;
  6. envoyer ces données sur Le Trameur pour étudier les cooccurrents spécifiques dans votre corpus.

Le script de récupération de pages

Ce script aura plusieurs objectifs, qu'on implémentera au fur et à mesure :

  1. lire une liste d'URL ;
  2. pour chaque URL, vérifier si elle est valide ;
  3. pour chaque URL valide, vérifier la réponse du serveur pour savoir si la page est accessible ;
  4. pour chaque page accessible, récupérer son encodage ;
  5. récupérer son contenu textuel et convertir au besoin en UTF-8 ;
  6. écrire dans un tableau différentes informations, dont :
    1. le code HTTP de retour ;
    2. l'URL elle-même ;
    3. le support (nombre d'occurrences) du mot cible dans la page ;

Ce contenu est sur une bande faisant la moitié de la largeur de l'écran, centrée.
